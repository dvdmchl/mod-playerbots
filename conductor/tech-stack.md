# Tech Stack: mod-playerbots

## Core Technologies
- **Programming Language:** C++ (Standard: C++17 or later as required by AzerothCore)
- **World of Warcraft Emulation:** AzerothCore (WotLK branch)
- **Database:** MySQL 5.7+ or MariaDB 10.3+
- **Build System:** CMake

## Platform & Environments
- **Operating Systems:** Ubuntu (preferred), Windows (MSVC), macOS (Clang)
- **Containerization:** Docker & Docker Compose (experimental)

## Tooling & Standards
- **Source Control:** Git
- **Code Style:** AzerothCore C++ Code Standards (enforced via `.clang-format`)
- **CI/CD:** GitHub Actions (for cross-platform builds and style checks)
- **Development Shell:** Bash / Linux environment (for installation scripts)
